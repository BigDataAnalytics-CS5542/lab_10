# Data

Raw data files are not tracked in this repository due to file size. To run the project, download the datasets below and place them in `data/raw/`.

## Required Datasets

### Fatal Overdose Data (Primary)
- **Source**: CDC SUDORS Dashboard
- **Link**: https://www.cdc.gov/overdose-prevention/data-dashboards/sudors-dashboard/data/SUDORS-Fatal-Overdose-Data.xlsx
- **Format**: XLSX
- **Save as**: `data/raw/fatal_overdose.xlsx`

### Non-Fatal Overdose Data (Primary)
- **Source**: CDC DOSE Dashboard
- **Link**: https://www.cdc.gov/overdose-prevention/data-dashboards/dose-discharge-dashboard/data/DOSE_dx_Dashboard_02092026.xlsx
- **Format**: XLSX
- **Save as**: `data/raw/nonfatal_overdose.xlsx`

## Supplementary Datasets

### Florida Post-Overdose Drug Data
- **Source**: FDLE MEC Publications
- **Link**: https://www.fdle.state.fl.us/MEC/Publications-and-Forms

### Substance Use by Demographic
- **Source**: Monitoring the Future (Panel)
- **Link**: https://monitoringthefuture.org/data/panel/

### Alcohol Use Rate (High Schoolers)
- **Source**: Monitoring the Future (Prevalence)
- **Link**: https://monitoringthefuture.org/data/bx-by/drug-prevalence/#drug=""

## Processed Data

Processed/cleaned CSVs will be generated by the preprocessing pipeline and saved to `data/processed/`. These are also gitignored.
